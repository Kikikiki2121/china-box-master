# Wanted Sales AI

Telegram Mini App для помощи в продаже мотоциклов с использованием искусственного интеллекта.

## Технологии

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn UI + Radix UI
- **Icons**: Lucide React
- **AI**: OpenAI API (gpt-4o)
- **Deployment**: Vercel

## Функционал

1. **Главный экран**: Кнопка "Продать байк" и список прошлых генераций
2. **Шаг 1 (Фото)**: Загрузка фото мотоцикла
3. **Шаг 2 (Анализ)**: Автоматический анализ фото через GPT-4o для определения марки, модели, цвета и модификаций
4. **Шаг 3 (Детали)**: Форма с предзаполненными данными от AI + поля для года, пробега, цены и валюты
5. **Шаг 4 (Стиль)**: Выбор стиля описания (Агрессивный, Спокойный, Срочный)
6. **Результат**: Сгенерированное описание с переводами на русский, английский и вьетнамский языки

## Установка

### Предварительные требования

Убедитесь, что у вас установлен Node.js (версия 18 или выше).

**Установка Node.js:**
- Через официальный сайт: https://nodejs.org/
- Через Homebrew: `brew install node`

### Быстрая установка

Используйте автоматический скрипт установки:

```bash
cd wanted-sales-ai
./setup.sh
```

Скрипт автоматически:
- Проверит наличие Node.js
- Создаст файл `.env.local`
- Установит все зависимости

### Ручная установка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env.local` в корне проекта:
```env
OPENAI_API_KEY=your_openai_api_key_here
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

3. Добавьте ваш OpenAI API ключ в `.env.local`

4. Запустите dev сервер:
```bash
npm run dev
```

5. Откройте [http://localhost:3000](http://localhost:3000)

## Деплой на Vercel

1. Подключите репозиторий к Vercel
2. Добавьте переменную окружения `OPENAI_API_KEY` в настройках проекта
3. Деплой произойдет автоматически

## Локальный доступ через ngrok

Для доступа к приложению из интернета используйте ngrok:

1. Установите ngrok: `brew install ngrok` или скачайте с https://ngrok.com/download
2. Настройте authtoken: `ngrok config add-authtoken YOUR_TOKEN`
3. Запустите ngrok: `ngrok http 3000` (приложение работает на порту 3000!)
4. Используйте публичный URL из ngrok для доступа

Подробнее в файле `NGROK_SETUP.md`

## Использование

1. На главной странице нажмите "ПРОДАТЬ БАЙК"
2. Загрузите фото мотоцикла
3. Дождитесь анализа (автоматически определится марка, модель, цвет)
4. Заполните детали (год, пробег, цена, валюта)
5. Выберите стиль описания
6. Получите готовое объявление на трех языках
7. Скопируйте текст и используйте для продажи

# Настройка OpenAI API ключа

## Проблема

Если вы видите ошибку "Ошибка при анализе фото" или "Ошибка при генерации описания", скорее всего проблема в том, что не настроен OpenAI API ключ.

## Решение

1. Получите API ключ OpenAI:
   - Перейдите на https://platform.openai.com/api-keys
   - Войдите в свой аккаунт (или создайте новый)
   - Нажмите "Create new secret key"
   - Скопируйте ключ (он начинается с `sk-`)

2. Откройте файл `.env.local` в корне проекта `wanted-sales-ai/`

3. Замените строку:
   ```env
   OPENAI_API_KEY=your_openai_api_key_here
   ```
   
   На:
   ```env
   OPENAI_API_KEY=sk-ваш-реальный-ключ-здесь
   ```

4. **ВАЖНО**: Перезапустите dev сервер после изменения `.env.local`:
   - Остановите сервер (Ctrl+C в терминале)
   - Запустите снова: `npm run dev`

## Проверка

После настройки API ключа:
1. Откройте http://localhost:3000
2. Нажмите "ПРОДАТЬ БАЙК"
3. Загрузите фото мотоцикла
4. Анализ должен пройти успешно

## Примечания

- API ключ хранится только локально в файле `.env.local`
- Файл `.env.local` не должен попадать в git (он уже в `.gitignore`)
- Убедитесь, что у вас есть доступ к модели GPT-4o в вашем OpenAI аккаунте
- Проверьте баланс на вашем OpenAI аккаунте

## Стоимость

Использование GPT-4o для анализа изображений платное. Примерная стоимость:
- Анализ одного фото: ~$0.01-0.05
- Генерация описания: ~$0.01-0.03

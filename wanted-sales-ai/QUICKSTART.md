# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
cd wanted-sales-ai
npm install
```

## Шаг 2: Настройка переменных окружения

Создайте файл `.env.local` в корне проекта:

```env
OPENAI_API_KEY=sk-your-api-key-here
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

**Важно**: Получите API ключ OpenAI на https://platform.openai.com/api-keys

## Шаг 3: Запуск приложения

```bash
npm run dev
```

Откройте браузер и перейдите на http://localhost:3000

## Шаг 4: Использование

1. На главной странице нажмите "ПРОДАТЬ БАЙК"
2. Загрузите фото мотоцикла
3. Дождитесь автоматического анализа
4. Заполните детали (год, пробег, цена)
5. Выберите стиль описания
6. Получите готовое объявление на 3 языках!

## Структура проекта

- `app/page.tsx` - Главная страница
- `app/sell/page.tsx` - Процесс создания объявления
- `app/result/[id]/page.tsx` - Страница результата
- `app/api/analyze/route.ts` - API для анализа фото
- `app/api/generate/route.ts` - API для генерации описания

## Деплой на Vercel

1. Загрузите код в GitHub
2. Подключите репозиторий к Vercel
3. Добавьте переменную окружения `OPENAI_API_KEY`
4. Деплой произойдет автоматически!

## Troubleshooting

**Ошибка "OPENAI_API_KEY is not defined"**
- Убедитесь, что файл `.env.local` создан в корне проекта
- Перезапустите dev сервер после создания файла

**Ошибка при анализе фото**
- Проверьте, что у вас есть доступ к GPT-4o в вашем OpenAI аккаунте
- Убедитесь, что API ключ правильный и имеет достаточный баланс

**Стили не применяются**
- Убедитесь, что Tailwind CSS правильно настроен
- Проверьте, что `tailwind.config.ts` существует и правильно настроен

# Исправление: ngrok на правильном порту

## Проблема
Вы запустили ngrok на порту 80, но приложение Next.js работает на порту **3000**.

## Решение

1. **Остановите текущий ngrok:**
   - Нажмите `Ctrl+C` в терминале, где запущен ngrok

2. **Убедитесь, что dev сервер запущен:**
   ```bash
   cd ~/Desktop/my-sky-app/wanted-sales-ai
   npm run dev
   ```
   Должно быть: `- Local: http://localhost:3000`

3. **Запустите ngrok на правильном порту:**
   ```bash
   ngrok http 3000
   ```

4. **Используйте новый URL:**
   - ngrok покажет новый URL вида: `https://xxxx-xxxx-xxxx.ngrok-free.app`
   - Этот URL будет работать с вашим приложением на порту 3000

## Текущий статус
- ✅ ngrok установлен
- ✅ authtoken настроен
- ⚠️ Нужно перезапустить ngrok на порту 3000
